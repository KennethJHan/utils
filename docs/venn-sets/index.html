참고
  • 집합이 두 개 미만이면 다이어그램 대신 안내만 표시됩니다.
  • 입력을 바꾸면 선택 항목 창은 비워집니다.
  • 동일한 문자열이 여러 집합에 있으면 교집합에 포함됩니다.
  • 비율(%)은 고유 항목 전체 수(각 문자열은 한 번만 센 합계)에 대한 해당 영역 개수 비율입니다.
  • 다이어그램은 원을 다각형으로 근사해 영역을 계산합니다. 모양은 데이터와 무관하게 고정입니다.
