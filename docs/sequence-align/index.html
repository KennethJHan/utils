서열 정렬

두 문자열에 대해 Needleman–Wunsch 전역 쌍대 정렬을 수행합니다. 갭은 -로 표시합니다. 결과는 열마다 색으로 구분되며, 가운데 줄은 일치 |·불일치 :로 표시합니다.

공백·쉼표는 무시하고, UTF-8 한 글자(코드포인트) 단위로 맞춥니다.

갭 점수는 보통 0 이하로 두는 것이 일반적입니다.

일치 불일치 갭 가운데 줄: | 일치, : 불일치, 갭 열은 공백

알고리즘
  • 전역 정렬: 두 서열 끝까지 맞추며, 갭 삽입으로 최적 점수를 찾습니다.
  • 동일한 최고 점수의 정렬이 여러 개일 수 있습니다. 역추적 시 대각 → 위 → 왼 순으로 택해 하나만 보여 줍니다.
  • 길이가 각각 수천 이상이면 브라우저가 잠시 멈출 수 있습니다. (대략 글자 수 곱이 너무 크지 않게 쓰는 것을 권장합니다.)
