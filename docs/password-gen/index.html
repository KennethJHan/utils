비밀번호 생성기

브라우저 안에서만 임시 비밀번호를 만듭니다. crypto.getRandomValues로 난수를 뽑으며, 서버로 보내지 않습니다.

4~128자. 여러 자리 숫자(예: 12)를 칠 때 1·2처럼 중간 단계가 그대로 유지됩니다. 비워 두고 포커스를 빼면 12로 돌아갑니다. 범위 밖이면 「생성」 시 안내합니다.

포함할 문자 종류

공백·따옴표·백틱·역슬래시 등은 일부 사이트에서 문제가 되어 제외했습니다. 최소 한 종류는 켜 두어야 합니다.

안내
  • 난수는 Web Crypto를 사용합니다. 이 페이지는 비밀번호를 저장하지 않습니다(브라우저 저장소 미사용).
  • 새로고침하면 결과가 비어 있을 수 있습니다. 중요한 비밀번호는 비밀번호 관리자에 넣는 것을 권장합니다.
  • 사이트가 특수문자를 거부하면 특수문자 체크를 끄거나 길이·종류를 조정해 보세요.
  • 선택한 문자 종류마다 최소 1글자씩 들어가므로, 길이는 선택한 종류 수 이상이어야 합니다.
