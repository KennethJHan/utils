시간

  • 시간 차 계산기 YYYY-MM-DD HH:MM:SS 또는 HH:MM:SS 두 시각의 차이 (B − A)
  • 시간 단위 변환 초 ↔ 분·초 ↔ 시·분·초 ↔ 일·시·분·초 (입력 시 서로 자동 맞춤)
  • 달리기 페이스 km/h ↔ km당 분·초 ↔ 5km·10km·하프·풀 예상 완주 시간 (한쪽 수정 시 나머지 동기화)

계산

  • 퍼센트 증감 계산기 A → B로 바뀔 때 몇 % 증가·감소인지 ((B − A) ÷ A × 100)
  • 연봉·소득세(구간) 연봉·세전·세후 월급 상호 환산, 통계청 평균 임금·소득 하위 70% 건보 참고표, 과세표준 상세

데이터 · 시각화 집합, 통계, 그래프
