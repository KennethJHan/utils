시간 단위 변환

같은 길이를 초 · 분+초 · 시+분+초 · 일+시+분+초로 서로 바꿉니다. 한쪽을 고치면 나머지가 맞춰 갱신됩니다.

총 초(정수·소수)

분 · 초
:

총 분 × 60 + 초 (표시는 0–59초로 맞춤)

시 · 분 · 초
:
:

시×3600 + 분×60 + 초 (표시는 시·분·초 각각 정규화, 시는 누적 시간)

일 · 시 · 분 · 초
:
:
:

일×86400 + 시×3600 + 분×60 + 초 (일 아래 시·분·초는 하루 안에서 정규화)

동작
  • 어느 칸이든 입력이 바뀌면, 유효한 숫자로 읽을 수 있을 때 총 초로 환산한 뒤 네 가지 표현을 모두 갱신합니다.
  • 분·초는 (분)×60+(초), 시·분·초는 (시)×3600+(분)×60+(초), 일·시·분·초는 일×86400+시×3600+분×60+초로 합산합니다. 빈 칸은 0으로 봅니다.
  • 한 그룹의 값을 모두 지우면 나머지 입력도 비웁니다.
